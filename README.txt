How to preview locally:
1. Open index.html in a browser (double-click or use Live Server extension).
2. The contact form stores leads in the browser's localStorage (prototype).

How to host on GitHub Pages:
1. Create a new GitHub repository (public or private with GitHub Pages enabled).
2. Upload all files (index.html, styles.css, script.js) to the repository root.
3. In repository Settings -> Pages, set Build source to 'main' branch / root and save.
4. Your site will be published at https://<your-username>.github.io/<repo-name>/
